## Supported Runtime Script commands in Moon OS
prt, let, slp, inp, rnd
add, sub, mul, div, mod
jmp, jeq, jne, jgt, jlt
int, str, typ, tim
len, pol, pop, psh
put, get, key, del
clr, drw, pxl
def, end, ret, cal
ife, ifg, els, fin

~ not supported:
for, nxt
prs, lod, sav

~ only in Moon OS:
lib <API> [$V*] [N?]
